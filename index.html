Reflection Statement

Parallax scrolling was something new to me. I had no experience with this or even knew what it was prior to this course. Along with the videos I watched and other online references, I was able to code this page and learn something new. Parallax scrolling creates an illusion of depth on a 2D screen by allowing elements to move independently, thereby giving your site a 3D feel. I began my project with the basic layout page used in the last assignment, and made some changes to the code in Notebook++ so I could see it as it all came together. There is a lot that a person can do with parallel scrolling, but in my research, it advises that a designer use this option wisely. Some sites can overdo parallax scrolling, making it annoying to those visiting the site.
